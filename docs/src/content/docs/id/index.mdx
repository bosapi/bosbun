## Mulai Cepat

```bash
bunx bosbun create my-app
cd my-app
bun run dev
```

Buka [http://localhost:9000](http://localhost:9000) dan mulai membangun.

## Tech Stack

| Lapisan    | Teknologi        |
| ---------- | ---------------- |
| Runtime    | Bun              |
| HTTP       | ElysiaJS         |
| UI         | Svelte 5 (Runes) |
| CSS        | Tailwind CSS v4  |
| Bundler    | Bun.build        |
